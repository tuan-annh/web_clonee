# React + TypeScript + Vite

**1. Cài package với yarn:**
- Lưu ý: cài yarn trong cmd trước khi sử dụng yarn:

```
  npm install yarn
```
- Sau đó dùng `yarn` hoặc `yarn install` để cài packages

```
  yarn install
```

**2. Chạy vite app lên localhost:**
```
  yarn dev
```
